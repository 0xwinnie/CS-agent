# 🤖 CS Agent - Intro for Lordcha

## What is CS Agent?

CS Agent is your **AI-powered community guardian** for SNS Discord. Not just a bot with canned responses—it's a teammate with memory, personality, and genuine care for the community.

---

## 🌟 Key Features

### 1. Personality System (Soul-Driven)
- Warm, helpful, but has boundaries
- Speaks naturally (not robotic)
- Uses humor appropriately
- Adapts tone based on user relationship

### 2. Memory System
- **Remembers users**: Knows who's new vs. regulars
- **Trust levels**: Auto-promotes from Stranger → Observed → Trusted
- **Conversation history**: Doesn't ask the same questions twice
- **Issue tracking**: Remembers previous problems users had

### 3. User Recognition
| Level | Badge | Access |
|-------|-------|--------|
| 🌟 Trusted | Core members | Full features |
| 👀 Observed | Regulars | Standard access |
| 🆕 Stranger | First-timers | Limited, watched |
| 🚫 Banned | Trolls | No access |

### 4. Proactive Behaviors
- **Greets newcomers** with welcome DM
- **Remembers context**: "Hey Alice! Last time you asked about DNS—did it work?"
- **Auto-detects issues**: Collects feedback without users typing `!feedback`
- **Daily reports**: Morning brief on what happened overnight

### 5. Smart Language Support
- User asks in Chinese → Replies in Chinese
- User asks in English → Replies in English
- **Default**: English for global community
- **Daily reports**: Always in English (work language)
